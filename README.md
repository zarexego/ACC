
ACC — это консольный Linux-лаунчер, предназначенный для быстрого запуска браузеров в приватном режиме, входа через локальный аккаунт и открытия популярных сайтов из удобного текстового меню.
Проект создан как лёгкий и быстрый инструмент для работы в терминале без графического интерфейса. Он подходит для пользователей Linux, которым нужен простой способ запускать браузерные сессии с приватностью и минимальным количеством лишних действий.
Возможности
Запуск через консольный интерфейс.
Вход и регистрация локального аккаунта.
Быстрое открытие популярных сайтов.
Запуск браузеров в приватном режиме.
Поддержка нескольких браузеров.
Резервный способ открытия ссылок через xdg-open.
Простой и минималистичный интерфейс без лишних зависимостей.
Поддерживаемые браузеры
Проект рассчитан на запуск через:
Firefox.
Tor Browser.
Brave.
Любой браузер через xdg-open, если основной вариант недоступен.
Как это работает
После запуска программа показывает текстовое меню, где пользователь может:
войти в аккаунт;
зарегистрироваться;
выйти;
открыть нужный сайт;
выбрать браузер для запуска в приватном режиме.
Если нужный браузер не найден, используется запасной способ открытия через системную команду.
Для кого этот проект
ACC подойдёт:
пользователям Linux;
тем, кто часто работает через терминал;
тем, кому нужен быстрый запуск приватного браузинга;
тем, кто хочет простой локальный лаунчер без сложной оболочки.
